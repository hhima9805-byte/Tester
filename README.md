# Mafia Interactive Business Card

An interactive, animated business card with dark red Mafia theme, featuring multiple engaging animations and effects.

## Features

### Main Elements
- **3D Animated "MAFIA" Title**: Large, bold letters with gradient effects and 3D depth
- **Avatar Placeholder**: Circular placeholder ready for profile image
- **Social Media Links**: Telegram, GitHub, and Reddit with custom SVG icons
- **Card Dimensions**: 700px wide, fully responsive

### Required Animations (3)

1. **Falling Snow**: Snowflakes appear and fall when hovering over the card
2. **Card Highlight**: Red glow effect on card borders and shadows on hover
3. **Shake/Vibration**: Subtle trembling animation when hovering

### Additional Animations (9)

4. **Avatar Pulse**: Avatar scales and glows rhythmically on hover
5. **Gradient Flow**: Animated gradient flowing through the MAFIA text
6. **3D Title Rotation**: Title rotates in 3D space on hover
7. **Sparkle Sweep**: Light sweeps across the title text
8. **Social Link Transform**: Links rotate 360Â° and scale up on hover with ripple effect
9. **Floating Social Icons**: Icons gently float up and down continuously
10. **Glowing Corners**: Corner decorations glow brightly on hover
11. **Ripple Effect**: Click anywhere on the card to create ripple effects
12. **3D Parallax**: Card follows mouse movement with 3D perspective

### Design Features
- Dark red color palette (#8b0000, #dc143c, #ff0000)
- Multiple shadow layers for depth
- Animated background particles
- Smooth transitions and cubic-bezier easing
- Fully responsive design for mobile devices

## Usage

Simply open `mafia-card.html` in any modern web browser. The card is fully self-contained with embedded CSS and JavaScript.

### Customization

To customize the card:
- **Avatar**: Replace the `.avatar-placeholder` background with an image
- **Social Links**: Update the `href` attributes in the social links section
- **Colors**: Modify the color values in the CSS variables
- **Animations**: Adjust animation durations and delays in the `@keyframes` sections
